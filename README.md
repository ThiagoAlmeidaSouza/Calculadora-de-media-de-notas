# Calculadora de média de notas + Aprovado ou Reprovado
Uma simples calculadora de média de notas, aplicando conceitos que aprendi na faculdade.

# O que foi usado?
Java puro.

# Como funciona?

Você diz seu nome;
Suas três notas;
E pronto! 

 Você receberá algo assim:

 Thiago, sua média das notas foram 8!
 Você foi Aprovado!
